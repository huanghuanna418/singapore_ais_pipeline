How to use this

  1. Pick an operator above and click Open.
  2. Upload that operator's Trip Report (.xlsx).
  3. Click Resolve MMSIs. Fill in any missing MMSIs by hand (in vessel_mmsi.csv).
  4. Run python 02_record_ais.py --operator <key> in a terminal and let it collect AIS data for as long as you can (days to weeks).
  5. Click Build Continuous Report.
  6. Click Estimate Energy.
  7. Hand the resulting Continuous Report to Estee's Flexsim code.

Local pipeline. Data stays on this computer.
